# Criação de Menu com Flexbox :page_facing_up:


``` HTML5 e CSS3 - nível básico ```

### Modelo: [Figma do exercício com Flexbox do curso DevQuest](https://www.figma.com/file/DYk9DZr6urB9MZ4iNt1a61/Desafio-HTML-%2B-CSS?node-id=0%3A1&t=xtR7eftqFCRx5TVo-0)

#### Conteúdo integrado:

- Declaração de tipo de documento;
- Idioma da página;
- Compatibilidade básica de navegadores;
- Declaração básica de responsividade;
- CSS3 (Folhas de estilo em cascata): Inline, Interno e Externo;
- Inserção do CSS3 no HTML5: link rel="stylesheet" href="style.css";
- Folhas de estilo independentes para reset e responsividade;
- Títulos da página;
- Reprodução de layout com design no Figma;
- Responsividade: "@media(max-width: tamanho) { conteúdo; }";
- Inserção de tipografia externa: Google-fonts;
- Flexbox no CSS: "Display: Flex";
- Características: Alinhamento de conteúdo, de itens, padrões de expansão, encolhimento, base, linhas, colunas...;
